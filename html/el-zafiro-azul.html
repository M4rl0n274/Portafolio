El Zafiro Azul

Este es un E-commerce con una interfaz atractiva y diseño responsivo, enfocado en optimizar las ventas mediante una experiencia de usuario fluida e intuitiva. El proyecto incluye un registro de usuarios y una conexión a bases de datos para gestionar eficientemente los productos disponibles. Este fue un proyecto colaborativo, en el cual se trabajó en equipo para integrar diversas tecnologías, logrando una tienda en línea funcional y moderna. La colaboración permitió combinar habilidades y enfoques para alcanzar un desarrollo de alta calidad.

Tecnologías

  • HTML/CSS: Para la estructura y el diseño adaptable de la interfaz de usuario.
  • JavaScript: Funcionalidades interactivas del carrito de compras y mejora de la experiencia del usuario.
  • Java: Lógica del backend y conexión segura con la base de datos.
  • MySQL: Gestión y almacenamiento eficiente de la información de usuarios y productos.
